WEBSITE LEMBAGA DESA

Website ini dibuat oleh mahasiswa KKN sebagai media informasi
dan dokumentasi kegiatan lembaga desa.

Isi website:
- Profil lembaga
- Struktur organisasi
- Kegiatan
- Kontak

Cara mengedit:
1. Buka file HTML
2. Edit teks di dalam <p>...</p>
3. Gambar disimpan di folder assets/img

Website ini bersifat statis dan dapat dikembangkan lebih lanjut.
